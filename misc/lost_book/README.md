Archaeologists recovered a corrupted hard drive from an ancient Minecraft Alpha server hosted in a basement in 2010. Rumor has it that the "Golden Apple Order" hid their vast treasury coordinates on this server.

We have extracted the playerdata of the Order's Grand Scribe. He was known to carry a heavy burden—and a heavier book. Decoding the file format is ancient history, but reading the contents might lead us to the gold.
