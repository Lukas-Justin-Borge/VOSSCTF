# Here is a list of useful tools: 


### Using Linux is smart, some challenges "require" it.

On windows: 

1. Open PowerShell as administrator

2. run `wsl --install`

3. Enjoy your small linux VM on windows B)

### Steganography (hidden stuff in images): 
https://georgeom.net/StegOnline/upload 
- Use this if you want to find hidden info within images. You can browse through the color planes and stuff like that

### Swiss knife/multipurpose tool: 
https://cyberchef.io/
- Here is a really good multipurpose tool where you can convert text/data to and from different formats and stuff. Very useful.

### Geohash: 
https://geohash.softeng.co/
- See geohash on a map.

http://geohash.co/
- Convert from latitude, longditude coordinates to geohash.

### Historical sattelite imagery:
https://earth.google.com/
- Can view historical timeline of sattelite images. 

### See older versions of websites
https://web.archive.org/
- You paste a link, and the website gives you snapshots from different dates.

### Wireshark
https://www.wireshark.org/download.html
- Packet inspection tool. Needed to insect `.pcapng` files.
